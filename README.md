# soothsayer / fortune teller board notes

- 1 pin in for button, watch for press + hold
- 2 pin outs for eye LEDs
- ??? for LCD display 

display pins
SCK = pin D8 or P2
MISO = D9 or P4
MOSI = D10 or P3
CS = pin D7 or P1
DC = pin D5 or P7
RST = pin D4 or P6

button pin = D0 or P26
led 1 = D1 or P27
led 2 = D2 or P28
